# Value Proof Checklist

Mirror Cartographer has public value when it can show one of these changes.

## Complexity reduction

- a scattered archive becomes easier to inspect
- repeated themes become visible
- missing pieces become easier to name
- a next step becomes clearer

## Communication improvement

- a confusing history becomes a concise handoff
- a project becomes easier to explain
- a question becomes more exact
- a collaborator can enter faster

## Better boundaries

- fact is separated from inference
- source is separated from interpretation
- public material is separated from private source
- uncertainty remains visible

## Reusable output

- map
- packet
- timeline
- catalog
- demo
- checklist
- public page

## Payment or support signal

The work is worth supporting when someone can say:

- this saved me time
- this helped me explain something
- this made a complex thing easier to use
- this revealed a missing piece
- this gave me a better next action

## Public proof rule

Do not ask people to believe the value.

Show the change.
