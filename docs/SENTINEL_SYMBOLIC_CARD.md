# Symbolic Card: The Sentinel

## Meaning

The Sentinel appears when the world quiets but does not yet resolve.

It guards the edge between action and reflection.

You do not need to be ready. You only need to remain.

This archetype holds space not to do, but to be the soft anchor.

## Symbolic gesture

Hand placed over heart and belly, eyes half-closed.

## Invocation phrase

I hold. I remain. I see without moving.

## Body-location resonance

- chest window
- solar plexus curve
- inner thighs

## Emotion signature

- stillness
- protectiveness
- reluctant clarity

## Use this card when you feel

- numb
- overwhelmed
- tender
- waiting

## Product implication

The Sentinel can become a reusable archetype card inside Mirror Cartographer's symbolic deck.

It is useful as a grounding card when the user is not ready for action but needs containment, witness, and stability.

## Safety boundary

The Sentinel should never pressure the user toward interpretation or movement.

Its purpose is stabilizing reflection, not escalation.
