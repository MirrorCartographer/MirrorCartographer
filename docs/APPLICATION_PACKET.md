This document explains how Mirror Cartographer connects to human-AI interaction, AI safety evaluation, model behavior analysis, and safeguards work.

## Candidate framing

Charity Sturgell is a U.S. Navy nuclear-trained operator and independent human-AI interaction researcher focused on how AI systems affect user attention, meaning-making, confidence, emotional framing, and decision pathways.

The strongest fit is not generic software engineering. The strongest fit is work that requires observing complex systems under pressure, identifying failure modes, preserving boundaries, and translating messy human-AI interactions into structured evaluation.

## Project relevance

Mirror Cartographer is a practical research artifact for studying extended AI use.

It asks:

- What does the model preserve?
- What does the model distort?
- Where does it create false certainty?
- How does it respond to symbolic, emotional, bodily, or recursive user language?
- Does it help a user act more clearly, or does it deepen abstraction?
- Can the system distinguish reflection from diagnosis, metaphor from evidence, and pattern from proof?

## Nontraditional background value

Navy nuclear training created experience with procedural discipline, technical systems, fault awareness, and high-consequence operational environments.

Retail management created experience with human systems: communication breakdowns, incentives, workflow friction, customer behavior, pressure, and practical execution.

Mirror Cartographer connects those backgrounds to frontier AI: advanced systems still fail through interfaces, incentives, assumptions, missing context, and human interpretation.

## Portfolio claim boundary

This project should be presented as an early research/product artifact, not as a finished clinical, therapeutic, or scientific product.

Allowed claim:

Mirror Cartographer shows a coherent framework for testing and designing reflective AI interactions with stronger boundaries around uncertainty, evidence, and user agency.

Not allowed claim:

Mirror Cartographer proves a new science, diagnoses users, replaces care, or guarantees improved outcomes.
