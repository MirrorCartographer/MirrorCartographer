# Five-Category Archive: Body, Style, Build, Map, Signal

Revision note:

- Status: source-derived and AI-organized category reference.
- Reason: created to preserve an early Mirror Cartographer archive structure from old conversations and make it searchable in GitHub.
- Source: prior conversation memory and project notes about organizing personal material, saved media, AI workflows, places, and signals.
- Boundary: this is an organizing system, not a claim that every future file must fit only one category.

## Overview

An early Mirror Cartographer archive structure used five core categories:

1. Body
2. Style
3. Build
4. Map
5. Signal

These categories were meant to organize life, projects, media, research, environment, and meaning without making the system too complicated.

## 1. Body

Body covers living systems and material care.

Includes:

- pets
- vet advice
- food
- symptoms
- sleep
- movement
- cleaning and home care
- routines
- sensory state
- nervous-system observations
- body-signal mapping

Mirror Cartographer use:

Body is where sensation becomes structured observation without becoming diagnosis.

## 2. Style

Style covers aesthetic expression and identity signals.

Includes:

- outfits
- makeup
- tattoos
- visual art
- music
- symbolic cards
- image references
- atmosphere
- color language
- visual identity

Mirror Cartographer use:

Style is not decoration only. It can be a signal layer for identity, mood, pattern, and public artifact design.

## 3. Build

Build covers making things real.

Includes:

- AI tools
- app ideas
- business ideas
- tutorials
- automation
- workflows
- coding
- GitHub
- Vercel
- product architecture
- evaluation artifacts

Mirror Cartographer use:

Build turns symbolic or emotional signal into interface, feature, proof, or artifact.

## 4. Map

Map covers location, movement, environment, and external world structure.

Includes:

- places
- cities
- restaurants
- travel
- relocation
- van/life logistics
- California / North Carolina comparison
- environment effects on body and skin
- home base decisions

Mirror Cartographer use:

Map links internal state to external environment and route planning.

## 5. Signal

Signal covers the meaning layer.

Includes:

- memes
- trends
- behavior
- recurring phrases
- typos
- symbolism
- dreams
- interaction patterns
- field observations
- emotional pressure
- repeated questions
- ceiling-breaking moments

Mirror Cartographer use:

Signal is where odd, repeated, or charged material is preserved, weighted, and tested before interpretation.

## Why the five categories matter

This archive structure keeps the project grounded in real use.

Mirror Cartographer is not only a theory system. It has to organize:

- animals
- body signals
- public artifacts
- software
- money path
- movement through places
- emotional and symbolic data
