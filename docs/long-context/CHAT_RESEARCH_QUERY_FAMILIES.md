# Chat Research Query Families

Revision note:

- Status: public-safe search strategy file.
- Reason: created so future research passes can search old chats more effectively using query families instead of one-off keywords.
- Source: old chat search passes, library results, user project-management instruction, and current organization work.
- Boundary: queries may locate private material. Any outputs committed to GitHub must be public-safe and abstracted.

## Purpose

A long chat archive cannot be researched well with one query.

Mirror Cartographer changed names, modes, and framing over time.

This file preserves reusable query families for future research passes.

## Output rule

A search hit is not automatically a file.

A search hit becomes useful only after it is:

1. abstracted if private
2. labeled by source status
3. labeled by claim status
4. routed to a folder
5. given a boundary
6. converted into a method, requirement, or research question
