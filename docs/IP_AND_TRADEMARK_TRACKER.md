## Filing-class planning

Potential USPTO classes to evaluate:

- Class 9: software, downloadable app, digital interface, AI-assisted tool
- Class 16: printed journals, cards, glyph decks, worksheets, manuals
- Class 41: education, workshops, courses, training, online learning
- Class 44: wellness or symbolic-support services, if carefully non-medical and properly described

## Filing-description drafts needed

Prepare descriptions for:

### Class 9

Software and digital interface for symbolic self-reflection, body-map input, pattern tracking, and AI-assisted reflective feedback.

### Class 16

Printed journals, symbolic cards, glyph decks, worksheets, and educational printed materials for structured symbolic reflection.

### Class 41

Educational services, courses, guided workshops, and online instruction in symbolic reflection, human-AI interaction, and self-mapping practices.

### Class 44

If used, this class requires extra care. Keep wording non-medical unless professionally reviewed.

Possible safer framing: wellness-support services involving symbolic reflection, non-clinical self-mapping, and guided reflective practices.

## Filing-method decision

Options:

1. DIY filing through USPTO.
2. LegalZoom or similar filing service.
3. Trademark attorney.

Tradeoff:

- DIY is cheapest.
- Filing service may reduce clerical burden.
- Attorney is strongest for strategy, classification, and office-action response.
