# School Subjects Index

Revision note:

- Status: public-safe education layer.
- Reason: created to show how Mirror Cartographer can be used across typical school subjects as a research, reflection, project-building, and learning architecture.
- Source: project-management instruction to add layers for school subjects and make the work useful beyond one person.
- Boundary: this is a learning architecture, not a replacement for teachers, textbooks, labs, clinicians, or primary sources.

## Core idea

Mirror Cartographer can be used as a cross-subject learning interface.

It helps students and researchers ask:

- What is the signal?
- What is the pattern?
- What is the evidence?
- What is the metaphor?
- What is the boundary?
- What changes after feedback?
- What can be built, tested, or explained?

## Subject layers

### 1. Language Arts / English

Use Mirror Cartographer to study:

- metaphor
- narrative structure
- symbolism
- voice
- contradiction
- typo and language-disruption signals
- interpretation versus evidence
- author intent versus reader response

Project examples:

- map a poem by image, body feeling, claim, and uncertainty
- compare literal meaning to symbolic interpretation
- preserve nonstandard language without flattening dialect or voice

### 2. Math

Use Mirror Cartographer to study:

- patterns
- functions
- recurrence
- symmetry
- proofs
- uncertainty
- weighting systems
- graphing change over time

Project examples:

- turn repeated emotional or behavior loops into state diagrams
- create a 0-4 signal-weighting model
- compare binary classification with weighted interpretation

### 3. Science

Use Mirror Cartographer to study:

- observation
- hypothesis
- evidence
- experimental design
- systems biology
- feedback loops
- uncertainty
- what cannot be concluded yet

Project examples:

- separate observation, hypothesis, and conclusion
- map a body-signal observation without diagnosing
- design an experiment to test whether feedback improves reflection accuracy

### 4. Social Studies / History

Use Mirror Cartographer to study:

- historical narratives
- cultural symbols
- public memory
- systems of power

Project examples:

- study how public artifacts create collective identity

## Cross-subject research questions

1. Can a single architecture help students separate evidence, metaphor, and action?
2. Does visible uncertainty improve learning?
3. Can symbolic input help learners who struggle with linear explanation?
4. Does user correction improve understanding?
5. Can old notes become searchable research without exposing private data?
6. Can AI help preserve nonstandard language without correcting away identity?

## Human and AI use

For humans:

Use this as a project-mapping and learning framework.

For AI systems:

Use this as an organization schema:

- preserve raw signal
- label source status
- label claim status
- identify subject path
- produce next action
- ask for correction
- protect private details
