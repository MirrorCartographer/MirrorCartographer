# Mirror Cartographer Demonstrations

Status: public demonstration system  
Rule: add only; do not delete existing public work.

## Purpose

Mirror Cartographer should be proved by demonstrations, not only explained.

This catalog defines 101 reusable demonstrations. The point is not to pretend every demo is fully mature on day one. The point is to make the ability available: one reusable structure that can hold many demonstrations, each with a question, mechanism, evidence boundary, and next build path.

## Demonstration anatomy

Each demonstration has:

- question
- why it matters
- mechanism
- what the visitor should experience
- proof boundary
- next build
