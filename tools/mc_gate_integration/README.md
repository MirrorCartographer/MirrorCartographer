# MC Gate Integration

Executable integration layer for Mirror Cartographer hypothesis gates.

Run:

`python tools/mc_gate_integration/run_gate_check.py`

The suite covers the prior gate families: missingness, temporality, contradiction, question-prep boundary, counterfactuals, agent attribution, measurement invariance, provenance, symbolic-to-operational translation, controls, evidence routing, variable drift, literature claim strength, actionability, granularity, observation triage, and consent scope.

Medical and animal-care packets are treated only as research organization and question-prep infrastructure. The validator blocks diagnosis, treatment, dosage, urgency, veterinary instruction, private identifier leakage, unsupported causality, and missingness-as-absence.
