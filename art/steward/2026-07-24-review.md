# Art Steward Run — 2026-07-24

## Art objective
Create a reusable visual model for the five active stewards that explains coordination without resembling a generic dashboard or decorative moodboard.

## Visual system decision
Use a radial system with a single coordination core. Each steward has equal orbital weight; the core is larger because it performs synthesis, not because it has greater authority. Evidence paths point inward to make the required sequence visible: specialized work becomes coordinated only after verification.

## Artifact produced
- `art/steward/steward-orbit.svg`
- Original editable SVG; no external imagery, fonts, characters, or protected brand assets.
- Includes semantic `title`, `desc`, labeled groups, and a reduced-motion-safe optional pulse.

## Accessibility review
- WCAG-oriented contrast was chosen for dark-background viewing.
- Every node is named in text; color is supplementary.
- The only animation is suppressed when `prefers-reduced-motion: reduce` is active.
- The SVG has an accessible title and long description.
- Remaining gap: provide a text-only adjacent explanation wherever the SVG is embedded.

## Rights and provenance
All geometry, copy, styling, and code are original for this repository. No external visual assets were used.

## Method experiment
**Weakness:** prior visual work frequently communicated atmosphere more strongly than operational structure.

**Experiment:** require every systems illustration to pass a three-channel meaning test: the relationship must remain understandable through (1) geometry, (2) labels, and (3) a plain-language description independently.

**Expected improvement:** at least 90% of evaluators can identify the central coordination role and all five steward roles without relying on color.

**Evaluation window:** next five visual-system artifacts.

**Status:** adopted provisionally; the SVG passes the internal rubric, but external comprehension testing has not yet been run.

## Risks and rollback
The radial arrangement may become cramped when additional stewards are added. Roll back by replacing the orbit with a layered flow diagram while retaining the same semantic labels and accessible description.

## Next executable step
Create a narrow-screen vertical derivative and a small HTML demonstration that lets a user select a steward and inspect its evidence contract without animation or hover-only interaction.
