# Art

This is the public gallery for art created during the Mirror Cartographer project.

It is intentionally separate from research and code.

Possible collections:

- Concept art
- Interface sketches
- Maps
- Album covers
- Posters
- Diagrams
- Typography
- Visual experiments
- Comedy Club artwork
- Proof films (storyboards)

Guiding idea:

Art here should make people curious before they understand.

Every piece should either:

- reveal a relationship,
- invite exploration,
- or make the project feel larger than a document.
