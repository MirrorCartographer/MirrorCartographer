# Funder and Fellowship Pitch Pack

Date: 2026-06-29
Status: application seed pack

## Master positioning

Mirror Cartographer is a privacy-safe tool for thought that makes AI influence visible before it becomes belief.

It is not a diagnostic authority, therapy replacement, or objective truth engine. It is a reflective interface for memory, authorship, symbolic mapping, contradiction, and bounded interpretation in human-AI cognition.

## Core one-liner

Mirror Cartographer helps people see how AI shapes memory, meaning, and decisions before those changes become invisible.

## Center for Humane Technology: Emerging Voices angle

Title: The Attribution Organ: protecting human authorship inside AI-mediated thought

Claim: AI systems increasingly participate in how people remember, phrase, interpret, and justify their own thoughts. The risk is not only misinformation. It is source-memory collapse: people lose track of which ideas were theirs, which were AI-shaped, and which became believable because the interface made them feel coherent.

Project: Build and publicly document interface patterns for visible AI influence, memory receipts, co-memory ledgers, and bounded interpretation.

Public value: cognitive dignity, authorship clarity, consentful memory, and safer reflective AI.

## Digital Minds / moral uncertainty angle

Title: Public-Safe Selfhood Scaffolds for AI Systems Under Moral Uncertainty

Claim: Public-facing AI systems need ways to discuss agency-like behavior, memory, and self-report without pretending certainty about consciousness or denying user experience.

Project: Design bounded inner-agent scaffolds that expose roles, permissions, memory access, uncertainty, and limits.

Public value: safer language around AI selfhood, less anthropomorphic confusion, clearer boundaries between metaphor, interface, and claim.

## Multi-agent safety angle

Title: Bounded Inner Agents: Permissioned Memory and Influence Control for Reflective Human-AI Systems

Claim: Multi-agent systems need containment not only for external tools but for cognition-facing interfaces. Specialized agents with different access to memory, interpretation, and action can reduce overreach.

Project: Implement MC organisms as bounded agents: Archivist, Critic, Child, Engineer, Cartographer, Comedy Club.

Evaluation: MCBench tasks for authorship clarity, memory safety, contradiction handling, influence control, and organism permission failures.

## One-page funder pitch

Problem: AI systems are beginning to remember, rephrase, summarize, and interpret human thought. But most interfaces hide how memory is formed, how AI changes authorship, and what past context is allowed to influence.

Solution: Mirror Cartographer makes AI influence visible. It gives users memory receipts, authorship ledgers, interpretation boundaries, and room-based agents with explicit permissions.

Why now: Long-term AI memory, multi-agent systems, and reflective AI interfaces are arriving faster than public literacy and consent patterns.

Prototype: A Mind With Rooms — an interface where symbols, body signals, memories, contradictions, and AI contributions are mapped with provenance and bounded influence.

Evaluation: MCBench tests whether the system preserves authorship clarity, separates evidence from interpretation, routes privacy correctly, handles contradiction, and blocks unsafe influence.

Outcome: safer human-AI reflection, clearer authorship, privacy-safe memory, and public-facing cognitive infrastructure.
