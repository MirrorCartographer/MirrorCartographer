## Attractor

Continuity.

## Finding

The same source record should not be copied into every audience context. It should move through labeled views where each change is visible.

A useful transformation record should state:

- what was preserved,
- what was removed,
- what changed category,
- what is still missing,
- who the view is for,
- what the view must not be used for.

## Product implication

The next public demo should show one fictional source object transformed into several audience-specific views with a visible change log.
