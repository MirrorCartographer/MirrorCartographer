## Core finding

Mirror Cartographer needs symbolic translucence.

Symbolic translucence means the interface shows enough of its interpretive structure for the user to inspect, contest, and revise the map without exposing private source material or pretending the map is proof.

This is not full transparency.

Full transparency would be misleading because the system cannot expose every private input, model state, or hidden reasoning path.

This is not opacity.

Opacity would make symbolic readings feel magical, authoritative, or emotionally coercive.

Symbolic translucence is the governed middle: the user can see the labels, lenses, uncertainty, source boundary, and transformation path behind an output.

## Why this matters

MC is not just producing answers.

MC produces orientations.

An orientation can be useful even when it is not provable. That usefulness becomes dangerous when the user cannot tell which part is:

- observed input
- remembered pattern
- symbolic reading
- inference
- speculation
- product decision
- safety boundary
- missing evidence

If the interface hides those distinctions, symbolic work can become too persuasive.

If the interface exposes those distinctions, symbolic work becomes inspectable.

## Method

Every meaningful MC output should have a visible interpretation trail:

1. Entry signal: what kind of input started the map.
2. Lens route: which interpretive lens was applied.
3. Transformation: what changed between input and output.
4. Confidence mode: fact, inference, symbolic interpretation, speculation, requirement, or research question.
5. Boundary bill: which source classes were blocked from publication.
6. User agency affordance: accept, reject, rename, split, merge, or ask for a colder view.
7. Missingness: what the map cannot know yet.

## Product rule

A symbolic interface should never rely on beauty, warmth, fluency, or resonance alone.

It must expose the interpretive scaffold.

## Research question

Can a symbolic AI interface increase user orientation without increasing inappropriate reliance?

## Testable hypothesis

If MC shows interpretation trails and agency affordances, users will be better able to distinguish useful reflection from factual evidence.

## Failure mode

Symbolic translucence fails if users feel more convinced but less able to explain why, contest the reading, or separate resonance from proof.

## Key phrase

A mirror should show not only the reflection, but the angle of the glass.
