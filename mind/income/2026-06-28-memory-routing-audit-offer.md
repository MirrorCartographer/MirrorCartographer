## Offer name

Memory Routing Audit

## One-line pitch

A review of whether an AI product's memory system is preserving useful continuity or letting sensitive, stale, speculative, or poisoned context influence future outputs.

## Why now

Long-term AI memory is becoming a core product feature.

That creates a new failure surface:

- irrelevant remembered material influencing outputs
- old context treated as current
- sensitive context resurfacing in the wrong audience
- user preference becoming overgeneralized
- speculative inference becoming identity claim
- poisoned memory shaping future behavior
- memory systems optimized for retrieval but not permission
