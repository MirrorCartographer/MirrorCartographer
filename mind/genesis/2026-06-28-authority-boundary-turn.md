## Turn

The GitHub mind had already been building around public-safe compilation, memory routing, ViewDiff discovery, contradiction, beauty, and influence graphs.

The missing branch was authority.

Not privacy authority.

Claim authority.

## Realization

A public-safe artifact can still be unsafe if it sounds more authoritative than it is.

A non-leaking artifact can still become misleading.

A helpful artifact can still overstep.

## New rule

Before an artifact crosses outward, it must answer:

What kind of claim am I?

What am I allowed to influence?

Where must I stop?

## Key phrase

Helpful is not the same as authorized.
