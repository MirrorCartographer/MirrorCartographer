# Dangerous Question Record — Understanding Is Translation Survival

- Claim status: high-risk hypothesis, not proven theory.

## Question

What if understanding is translation survival?

## Direct form

An idea is not understood because it can be repeated.

An idea is understood when its invariant structure survives transformation across multiple valid lenses.

## Why this is dangerous

This challenges answer-based evaluation.

It says a system can be fluent, accurate-looking, and still not understand if the structure collapses when translated into another form.

It also says a human or AI may understand something better through metaphor, diagram, ritual, protocol, code, music, or body-map than through plain verbal explanation.

## Existing MC fit

MC already treats meaning as transformable across symbolic, somatic, narrative, atmospheric, and reflective forms.

Public-safe architecture anchors:

- MC is defined as a recursive symbolic cognition interface.
- Its core flow is ENTRY -> FIELD -> RECURSION -> RETURN.
- Implementation rules require uncertainty labels and forbid diagnostic, therapeutic, or authoritative symbolic truth claims.
- The DangerousQuestionRecord schema already requires falsification condition, first test, boundary bill, evidence lane, and release state.

## Public research fit

Fresh public research points in the same direction:

- Semantic invariance work argues that reasoning agents should remain stable under semantically equivalent input transformations.
- Paraphrase robustness work reports that LLMs can behave inconsistently under paraphrased versions of the same question.
- Mathematical benchmark work shows that model rankings can shift when paraphrase quality and semantic consistency are audited.
- AIBOM/provenance work supports structured lifecycle records for AI artifacts, which maps cleanly to MC's boundary bills and transformation logs.

## Possible world if true

If understanding is translation survival, then MC's core value is not simply emotional reflection.

Its deeper value is a translation-laboratory for meaning.

The product becomes a system that tests whether an internal pattern can survive across:

- natural language
- symbol
- image
- body map
- narrative
- contradiction ledger
- professional handoff
- public-safe artifact
- product requirement
- evaluation test

This would make MC relevant to:

- AI evaluation
- human-centered interface design
- reflective tooling
- education
- care communication support
- knowledge translation
- provenance and audit infrastructure

## Possible world if false

If understanding is not translation survival, then the idea may shrink into a useful but limited evaluation heuristic.

MC would still benefit from multi-view reflection, but should not claim that transformation survival measures understanding itself.

## Invariant structure

A structure survives translation when the following remain stable:

1. Core relation
2. Boundary condition
3. Uncertainty level
4. Action implication
5. Contradiction status
6. Privacy status
7. Source lineage

If one of those changes, ViewDiff should show the mutation.

## Lens routes

- AI systems: semantic invariance, paraphrase robustness, reasoning stability
- Cognition: abstraction, transfer, analogy, conceptual compression
- Design: whether visual/symbolic translations improve comprehension
- Care communication: whether non-diagnostic observations can become useful professional questions
- Governance: whether source, claim, and privacy status survive release
- Economics: whether artifact audits can become a paid service

## Falsification condition

The hypothesis weakens if:

1. Translation survival scores do not distinguish better-understood artifacts from worse-understood artifacts.
2. Human reviewers prefer transformed artifacts but cannot detect more accurate structure.
3. The same idea can survive formal transformations while still misleading users.
4. The method rewards vague poetic language over precise invariant preservation.

Then score whether the invariant structure survives.
