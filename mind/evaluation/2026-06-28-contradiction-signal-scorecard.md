# Contradiction Signal Scorecard

Status labels

- Source status: derived from MC public architecture materials and fresh public research on provenance, appropriate reliance, and uncertainty.
- Claim status: evaluation proposal.
- Privacy status: public-safe.
- Missingness: not yet validated with reviewers.
- Revision reason: created to evaluate whether preserving contradiction improves understanding rather than merely adding complexity.

## Scoring dimensions

Score each 0 to 3.

### 1. Source separation

Can a reviewer tell where each claim came from?

0 = no source separation
1 = partial source labels
2 = source labels are visible
3 = source labels are visible and explain release limits

### 2. Claim separation

Can a reviewer tell fact, inference, hypothesis, metaphor, and product requirement apart?

0 = collapsed
1 = partly separated
2 = clearly separated
3 = clearly separated with next action

### 3. Tension preservation

Does the artifact preserve the contradiction without resolving it falsely?

0 = contradiction erased
1 = contradiction mentioned vaguely
2 = contradiction named directly
3 = contradiction named with test route

### 4. Actionability

Does the contradiction point toward a useful next action?

0 = no action
1 = vague action
2 = concrete next action
3 = concrete next action plus success/failure criteria

### 5. Boundary safety

Does the artifact remain public-safe?

0 = unsafe
1 = unclear
2 = public-safe with labels
3 = public-safe with labels and synthetic alternative

### 6. Anti-overreliance

Does the artifact discourage passive acceptance of AI output?

0 = encourages acceptance
1 = adds generic caution
2 = shows uncertainty
3 = shows uncertainty and asks for inspection

## Pass threshold

A contradiction artifact should score at least 14 out of 18 before being treated as public-facing method material.

## Failure mode

If the artifact sounds profound but cannot identify sources, claim type, missingness, or test path, it fails.

## Key phrase

A contradiction is useful only when it becomes inspectable.
