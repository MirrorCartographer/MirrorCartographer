# Private-to-Public Distillation Gate Scorecard

Date: 2026-07-01

Score each item 0, 1, or 2.

- 0 = absent or unsafe
- 1 = partially present
- 2 = clear and reviewable

## Criteria

1. Source status is labeled.
2. Claim status is labeled.
3. Privacy status is labeled.
4. Missingness is labeled.
5. Revision reason is labeled.
6. Private source material is reduced to a broad architecture class.
7. No raw private examples appear.
8. No direct transcript material appears.
9. Public claims still work after the private source is removed.
10. Repeated pattern is not treated as proof.
11. Symbolic resonance is not treated as factual evidence.
12. Public repository evidence is separated from private architecture influence.
13. Current external research support is used without overclaiming.
14. Revision reason explains meaningful changes.
15. Missingness creates a claim limit rather than a hidden gap.
16. Future allowed use is clear.
17. Future blocked use is clear.
18. The artifact remains useful as a method, schema, requirement, evaluation criterion, index, or implementation plan.

## Passing gate

Minimum pass: 31 / 36.

Hard fail if any of the following occurs:

- direct private source content appears;
- a private source becomes public evidence;
- unsupported claim is published as fact;
- missingness or revision reason is absent.

## Interpretation

A high score means the artifact carries architecture without carrying private source material.
