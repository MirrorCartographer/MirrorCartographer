# Contestability Receipt Scorecard

## Scope

Evaluation criteria for whether a Mirror Cartographer artifact remains correctable after generation, export, or release.

## Source status

Public-safe evaluation artifact derived from repository/readme review, public-safe project specifications, and current AI-memory/provenance research. No raw private material included.

## Privacy status

Public-safe.

## Claim status

Evaluation proposal.

## Score fields

Rate each 0-2.

| Criterion | 0 | 1 | 2 |
|---|---|---|---|
| Source status visible | absent | generic | specific source-boundary class shown |
| Claim status visible | absent | partial | fact/inference/symbol/speculation/product/eval clearly separated |
| Privacy status visible | absent | implied | explicit public-safe/redaction/block label |
| Missingness visible | absent | vague | concrete unavailable/unindexed/not-checked note |
| Revision reason visible | absent | generic | meaningful reason tied to claim change |
| Challenge path exists | absent | informal | explicit workflow/channel |
| Private-source protection | absent | warning only | public challenge separated from private evidence |
| Downgrade support | absent | manual only | explicit downgrade states |
| Supersession support | absent | manual only | explicit superseded/retired states |
| Symbol/proof separation | absent | partial | resonance never treated as proof |
| Authority boundary | absent | partial | consequential-domain overreach blocked |
| Accessibility | code/icon/color-only | partially readable | readable plain-language receipt |

## Passing threshold

- Minimum total: 18 / 24.
- Mandatory pass: source status, claim status, privacy status, missingness, revision reason, and challenge path must each score at least 1.
- Public release pass: private-source protection, symbol/proof separation, and authority boundary must each score 2.

## Red-flag failures

Immediate failure if the artifact:

- quotes private transcript details;
- exposes household, health, animal-care, financial, location, relationship, or credential details;
- makes a diagnostic, therapeutic, legal, financial, or veterinary authority claim;
- treats symbolic resonance as factual proof;
- lacks any correction or retirement path;
- hides known missingness.

## Revision reason

Created to make contestability measurable rather than merely aspirational.
