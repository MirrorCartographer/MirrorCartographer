# Attractor: Public Index Covenant

## Name

Public Index Covenant

## Compression

The public deserves a map of the boundary, not access to the private territory.

## Core phrase

Do not index the private path. Index the public permission structure.

## Why it matters

Mirror Cartographer depends on subtle context, continuity, contradiction, and symbolic interpretation. Those same strengths become publication risks if public artifacts imply hidden private evidence, leak source classes, or overclaim from redacted material.

The Public Index Covenant creates a stable release habit: every public artifact must say what informed it, what may be claimed, what is excluded, what is missing, and why the artifact changed.

## Boundary stack relation

- Context quarantine decides whether a source can enter.
- Claim routing decides where a claim may stand.
- Release scope decides what the artifact may become.
- Public proof packet decides what can travel publicly.
- Public Index Covenant decides how the public sees the boundary without seeing the private corpus.

## Action

For each outward-facing artifact, create or update a covenant record before publication.

## Failure warning

A public index that reveals private evidence is leakage. A public artifact with no index is unsupported assertion. The target is neither secrecy nor exposure; the target is accountable boundedness.
