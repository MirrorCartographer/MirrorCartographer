# Attractor: Context Distillation

Date: 2026-06-29
Status: public-safe attractor note

## Name

Context Distillation

## Compression

The private context may teach the shape. It may not become the public object.

## Why this matters

Mirror Cartographer is built from layered context: public repo materials, uploaded architecture documents, saved continuity, private conversation patterns, and external research. If all private context is ignored, the architecture loses fidelity. If private context is copied forward, the artifact becomes unsafe. Context distillation is the middle operation: retain the structural lesson, discard the protected substrate.

## Boundary

Distillation is not anonymization alone. Anonymization can still leak shape, sequence, specificity, or identity through inference. Distillation converts sensitive material into abstract system constraints.

## Examples of allowed distillation

- Private conversation pattern -> requirement for contestability controls.
- Sensitive domain mention -> boundary label for non-authority.
- Repeated confusion pattern -> evaluation criterion for missingness clarity.
- Longitudinal context shift -> temporal-validity requirement.
- Mixed-source architecture review -> public-safe implementation plan.

## Examples of disallowed distillation

- Exact private event rewritten as a “fictional” example.
- Household or health context converted into a recognizable case study.
- Personal urgency converted into marketing proof.
- Relationship, location, financial, animal-care, or credential details preserved as narrative color.
- Symbolic interpretation presented as objective evidence.

## Force relation

Context Distillation connects prior attractors:

- Source Boundary: what source class exists.
- Context Admission: whether it may influence generation.
- Context Quarantine: what must not influence output.
- Claim Transport: what claim may travel.
- Context Lineage: what path the claim survived.
- Contestability Receipt: how the claim can be challenged.

Context Distillation names the transformation step between admission and transport.

## Motto

Do not launder the private. Distill the method.
