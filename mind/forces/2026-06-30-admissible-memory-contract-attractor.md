# Attractor: Admissible Memory Contract

Status: force note
Privacy status: public-safe
Revision reason: added to keep future MC research runs from treating memory retrieval as automatic authority.

## Phrase

Relevance opens the archive. Admission decides whether the archive may speak.

## Pressure

MC keeps accumulating context, research notes, product requirements, and proof artifacts. Without a memory-admission contract, the system risks treating anything retrievable as usable. That creates privacy leakage, stale-state claims, symbolic overreach, and false proof-transfer.

## Direction

Move from:

- memory as recall
- similarity as relevance
- relevance as authority
- private context as public proof
- old summaries as current state

Toward:

- memory as permissioned source object
- similarity as retrieval candidate only
- admission as a boundary decision
- private context as architecture input only
- revision as custody

## Hard rule

No remembered source may support a public claim unless it passes source, privacy, claim, temporal, missingness, and revision checks.

## Design image

The archive is not a mouth. It is a witness box.
