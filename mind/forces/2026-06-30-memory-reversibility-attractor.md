# Force: Memory Reversibility

## Attractor

Continuity becomes dangerous when admission becomes permanent.

## Core finding

A memory is not safe because it was once admitted. It is safe only while its admission conditions still hold.

## Tension

Mirror Cartographer needs continuity to recognize repeated language, symbols, emotional patterns, feedback, contradiction, and evolving self-description. But persistent continuity can harden into false authority if old material never loses permission.

## Design resolution

Treat every memory-like item as conditionally admitted. Admission must include a path back out.

## Field rule

- Admit only inside a claim lane.
- Reuse only inside the same or weaker lane.
- Downgrade when context weakens.
- Revoke when privacy, trust, or source conditions fail.
- Preserve only the public-safe reason for the change.

## Public-safe slogan

Do not make memory loyal. Make memory accountable.

## Source status

- Public repo: bounded reflection and claim-boundary architecture.
- Private context: abstracted only into consent and privacy requirements.
- External research: memory search, poisoning, provenance, citation, and trust-boundary risks.

## Claim status

Product/research attractor; not empirical validation.
