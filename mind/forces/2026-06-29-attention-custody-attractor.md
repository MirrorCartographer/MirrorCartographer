# Attractor: Attention Custody

Status: public-safe force note

## Attractor

Attention Custody

## Core sentence

Before a system makes a claim, it has already chosen where to look.

## What it pulls together

- context admission
- source boundaries
- claim transport
- influence ledgers
- redaction fidelity
- release readiness
- resonance/proof separation
- agency friction
- boundary regression testing

## What it adds

It names the pre-claim layer: attention routing.

A public-safe system must not only ask whether the final claim is safe. It must ask whether the path of attention was authorized, bounded, and contestable.

## Failure mode

If attention custody is absent, the system can honestly label a final claim while hiding the private or inadmissible context that made that claim seem important in the first place.

## Design rule

A context class may shape attention only if its influence can be labeled without exposing protected material.

## Key phrase

The claim is not the first movement. Attention is.
