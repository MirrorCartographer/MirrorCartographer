# Force Note: Index Admissibility

Status: public-safe attractor note
Privacy status: abstracted; no private details
Revision reason: captures the strongest architectural lesson from this run.

## Attractor

Index Admissibility

## Core phrase

Do not index the memory. Index the permission to use it.

## What it corrects

Mirror Cartographer can become too connected if every remembered or retrieved pattern is allowed to influence every output. Connection creates continuity, but custody creates trust.

The index must not merely gather fragments. It must hold boundaries.

## Design pressure

A future MC memory viewer should show not only what was remembered, but what that memory is allowed to become:

- reflection
- hypothesis
- product requirement
- implementation plan
- evaluation criterion
- public claim
- blocked/private-only material

## Failure mode

The failure is not only leaking private data. The quieter failure is letting private, stale, generated, or symbolic material carry public authority it never earned.

## System rule

Retrieval is access. Admissibility is permission. Evidence is a narrower class than memory.
