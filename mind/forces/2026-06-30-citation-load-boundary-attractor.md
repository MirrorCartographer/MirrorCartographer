# Attractor: Citation Load Boundary

## Phrase
A cited source can still be asked to carry too much.

## Operating law
Do not only ask whether a source is real. Ask whether it can survive the claim placed on it.

## Why it matters
Mirror Cartographer handles symbolic, emotional, reflective, and memory-shaped material. That makes it easy for a source to feel relevant before it is actually sufficient. The citation-load boundary prevents public artifacts from turning relevance into validation, resonance into proof, or private architecture memory into hidden evidence.

## Public-safe use
Use this attractor when preparing README updates, public indexes, proof packets, whitepaper claims, evaluator notes, and demo explanations.

## Stop condition
If a claim cannot stand without private context, it must be narrowed, relabeled, moved private, or blocked.
