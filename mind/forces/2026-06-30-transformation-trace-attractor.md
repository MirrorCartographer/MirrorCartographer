Date: 2026-06-30

Claim status: conceptual design attractor
Revision reason: added to compress the research run into a stable design force for future MC builds

## Name

Transformation Trace

## Core sentence

**A private source may shape the map only after the private source has been made unnecessary to publish.**

## Attractor behavior

This force pulls MC away from two failures:

1. Extraction failure: private material leaks into public artifacts.
2. Amnesia failure: private-context learning is discarded because it cannot be published directly.

The useful middle path is transformation.

## Operating rule

Private pressure may become:

- a boundary rule;
- a product requirement;
- an evaluation criterion;
- a research question;
- an implementation plan;
- a missingness note;
- a revision reason;
- a privacy-safe index.

Private pressure may not become:

- a quote;
- a disguised transcript;
- hidden evidence;
- identity inference;
- diagnosis;
- authority claim;
- public biography;
- proof by resonance.

## Short test

Before publishing, ask:

Can the public artifact still work if the private source disappears?

If yes, the transformation is likely safe.
If no, the artifact may still be carrying private content as hidden evidence.
