# Boundary Stack Attractor

**Attractor:** Boundary Stack

**Phrase:** A boundary is not a sticker. It is a stack.

## Meaning

A single label can decorate an artifact without controlling it. A stack creates ordered custody: source first, privacy next, temporal status next, lineage next, evidence before belief, scope before release, contestability before public confidence.

## Pull

This attractor pulls MC away from isolated safety slogans and toward compositional public proof.

## Public-safe use

This phrase can be used in public documentation because it names a method, not a private source.
