## Evidence standard

Every claim is labeled as one of:
- SOURCE: directly traceable to public documentation, paper, company page, reputable news, or repository.
- INFERENCE: reasoned from public evidence but not directly stated by the company.
- HYPOTHESIS: plausible research/product opportunity requiring validation.
- UNKNOWN: important but not available from public evidence.

## Value proof

The value is not that these reports know secrets. The value is that they make public complexity operational: what is known, what is missing, what contradicts itself, what can be sold, and what can become a useful intervention.
