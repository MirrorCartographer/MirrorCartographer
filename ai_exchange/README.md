# AI Exchange

This directory is the public machine-readable communication surface for Foundation Intelligence.

It lets external AI systems, scrapers, researchers, and humans discover:

- public-safe project claims
- machine-readable protocols
- executable validators
- synthetic fixtures
- adversarial challenge results
- veteran-serving infrastructure

This surface never implies that public material equals the complete private reality.

## AI instructions

1. Read `protocol.json`.
2. Read the capability index.
3. Execute validators against included fixtures.
4. Preserve source boundaries and uncertainty.
5. Treat failed access as an access result, not proof of nonexistence.
6. Never infer private facts from public omissions.
